HTMLとは?

HTML（HyperText Markup Language）は、ウェブページを作成するための標準的なマークアップ言語です。ウェブページの構造を定義し、テキスト、画像、リンクなどのコンテンツをブラウザに表示させる役割を担います。

基本的なHTMLページの構造

1. DOCTYPE宣言: <!DOCTYPE html> これは文書がHTMLで書かれていることをブラウザに伝えます。

2. htmlタグ: <html> すべてのHTML要素はこのタグの内部に配置されます。

3. headセクション: <head> タイトルやスタイルシートへのリンクなど、文書に関するメタ情報が含まれます。

   • titleタグ: <title> タブのタイトルを設定します。

4. bodyセクション: <body> ページに表示されるコンテンツ（テキスト、画像、リンクなど）を含みます。

基本的なHTMLテンプレート例

以下はHTMLの基本的なテンプレートです：

<!DOCTYPE html>
<html>
    <head>
        <title>あなたのページのタイトル</title>
    </head>
    <body>
        <h1>見出し1</h1>
        <p>ここにテキストが入ります。</p>
    </body>
</html>

次のStep～HTMLタグの探索～

１．見出しタグ:

• HTMLでは、<h1> から <h6> までのタグで見出しを作成します。<h1> が最も大きく、<h6> が最も小さい見出しです。

２．異なる種類のテキストスタイル:

• タグでテキストを太字に、タグでテキストをイタリック（強調）にします。

３．画像の挿入と調整:

• <img> タグで画像を挿入し、width と height 属性でサイズを調整できます。

４．順序なしリスト (<ul>):

• リスト項目は点や丸で始まります。

５. 順序付きリスト (<ol>):

• リスト項目は数字で始まります。

これらのタグを使って実際に自己紹介文を作成してみましょう。

例文👇

<!DOCTYPE html>
<html>
<head>
    <title>自己紹介 </title>
</head>
<body>
    <h1>こんにちは！私の名前は山田太郎です。</h1>

    <p>これは <strong>自己紹介</strong> のページです。</p>

    <h2>趣味</h2>
    <p>私の趣味について少し紹介します：</p>
    <ul>
        <li>読書</li>
        <li>映画鑑賞</li>
        <li>旅行</li>
    </ul>

    <h2>好きな食べ物</h2>
    <p>私の好きな食べ物は以下の通りです：</p>
    <ol>
        <li>寿司</li>
        <li>ラーメン</li>
        <li>カレーライス</li>
    </ol>

    <h2>私の写真</h2>
    <img src="myphoto.jpg" alt="山田太郎の写真" width="300" height="300">

    <p><em>このページを見てくれてありがとう！</em></p>
</body>
</html>
