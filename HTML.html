HTML とは?

HTML（HyperText Markup Language）は、ウェブページを作成するための標準的なマークアップ言語です。ウェブページの構造を定義し、テキスト、画像、リンクなどのコンテンツをブラウザに表示させる役割を担います。

基本的な HTML ページの構造

  1. DOCTYPE 宣言: <!DOCTYPE html> これは文書が HTML で書かれていることをブラウザに伝えます。
  2. html タグ: <html> すべての HTML 要素はこのタグの内部に配置されます。
  3. head セクション: <head> タイトルやスタイルシートへのリンクなど、文書に関するメタ情報が含まれます。
    • title タグ: <title> タブのタイトルを設定します。
  4. body セクション: <body> ページに表示されるコンテンツ（テキスト、画像、リンクなど）を含みます。

基本的な HTML テンプレート例

<!DOCTYPE html>
<html>
 <head>
 <title>あなたのページのタイトル</title>
 </head>
 <body>
 <h1>見出し 1</h1>
 <p>ここにテキストが入ります。</p>
 </body>
</html>

練習

  1. テキストファイルを開きます。
  2. 上記のテンプレートをファイルに入力します。
  3. 入力し終わったら名前を付けて保存し、index.html という名前で保存します。
  4. index.html を開くと、あなたのウェブページを表示させることができます。

次の Step～HTML タグの探索～

１．見出しタグ

HTMLでは、<h1> から <h6> までのタグで見出しを作成します。これらのタグは見出しのサイズを異ならせ、ページの構造を明確にします。例えば、<h1> は最も大きい見出しで、<h6> は最も小さい見出しです。

<h1>これは h1 タグの見出しです</h1>
<h2>これは h2 タグの見出しです</h2>
<h3>これは h3 タグの見出しです</h3>
...
<h6>これは h6 タグの見出しです</h6>

２．異なる種類のテキストスタイル

<strong> タグと <em> タグは、テキストのスタイルを変更するために使います。例えば、<strong> タグはテキストを太字にし、<em> タグはテキストをイタリック（強調）にします。

<p>これは <strong>太字</strong> のテキストです。</p>
<p>これは <em>イタリック</em> のテキストです。</p>

３．画像の挿入と調整

<img> タグを使って画像をウェブページに挿入します。主要な属性には、画像の場所を指定する src、代替テキストを提供する alt、そして画像のサイズを調整する width と height があります。

基本的な使用法

<img src="path/to/image.jpg" alt="画像の説明" width="500" height="300">

この例では、「path/to/image.jpg」の場所にある画像を挿入し、幅500ピクセル、高さ300ピクセルで表示します。画像が読み込まれない場合には「画像の説明」というテキストが表示されます。

ヒント: alt 属性は、画像が表示されない場合やアクセシビリティのために重要です。また、画像のサイズは、ページの読み込み速度に影響するため、適切なサイズを選ぶことが大切です。

４．順序なしリスト (<ul>)

順序なしリストは、点や丸で始まるリスト項目を作成するのに使用します。各リスト項目は <li>（List Item）タグで囲みます。

<ul>
    <li>アイテム1</li>
    <li>アイテム2</li>
    <li>アイテム3</li>
</ul>

５. 順序付きリスト (<ol>)

順序付きリストでは、リスト項目が数字で始まります。順序なしリストと同様に、各項目は <li> タグで囲みます。

<ol>
    <li>最初のアイテム</li>
    <li>次のアイテム</li>
    <li>3番目のアイテム</li>
</ol>

自己紹介文の例

<!DOCTYPE html>
<html>
<head>
 <title>自己紹介 </title>
</head>
<body>
 <h1>こんにちは！私の名前は奈良太陽です。</h1>
 
 <p>これは <strong>自己紹介</strong> のページです。</p>
 
 <h2>趣味</h2>
 <p>私の趣味について少し紹介します：</p>
 <ul>
 <li>読書</li>
 <li>映画鑑賞</li>
 <li>旅行</li>
 </ul>
 
 <h2>好きな食べ物</h2>
 <p>私の好きな食べ物は以下の通りです：</p>
 <ol>
 <li>寿司</li>
 <li>ラーメン</li>
 <li>カレーライス</li>
 </ol>
 
 <h2>私の写真</h2>
 <img src="myphoto.jpg" alt="奈良太陽の写真" width="300" height="300">
 
 <p><em>このページを見てくれてありがとう！</em></p>
</body>
</html>
